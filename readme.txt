# テストとは

## 大きく３つの種類に

### １．単体テスト
関数・メソッド単位のテスト。
今回はココ

### ２．結合テスト
複数の関数・メソッドを使ったテスト。

### ３．システムテスト
実運用と同じく最初から最後までを通したテスト。


# 準備
pytestは標準機能でないのでインストールが必要。
pip install pytest

testsフォルダに以下を用意。
・ファイル名「__init__.py」
　中身は何もないファイルを用意。
・test_code.py
　実際にテストコードをここにかいていく。

テストコードを書いたら、
cd tests
pytest test_code.py 
とすると、うまくいけば下記のようになる。
